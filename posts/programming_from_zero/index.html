Programming from Zero

Posted on Jun 21, 2022

You could say that it has never been easier to learn how to program. Python tutorials abound, there is an endless number of free courses, an even more endless number of free books, et cetera, et cetera.

However, when I was first learning to write code, those tutorials did not tell me what I wanted to know. What I wanted to know was how a computer works, and what what does it mean to program it? I wanted context and understanding, not just to be told to type magic words and marvel at the fact that I can print numbers on a screen.

There is an ocean of resources for beginners on getting a computer to do what you want, but not much that provides any context for what your programs are actually doing. If you’re learning because you have a specific thing you want to make and that’s all you care about, then that’s fine. But for a person who is simply curious about how the machines we use everyday work, that’s not enough.

So, I am going to write the programming tutorial I wish I had when I was learning. I will teach you, from zero, the basics of how a computer works, and what ‘programming’ actually means. But since the topic of ‘how a computer works’ is deeper than the ocean, I am going to have to simplify a huge amount. You will get the big picture, and I will provide links to extra resources if there’s a particular area you want to dive into deeper.

This post is meant to teach, not just give you a tutorial, so it will be pretty long. I suggest you get a hot beverage.

How Does a Machine do Math?

We call it a computer because, first and foremost, it computes. It does math. ‘How?’, we ask. It’s actually not that complicated.

The Basics of Computer Architecture

A machine needs three basic components to be considered a ‘computer’. We’ll list them here and then go into detail:

  1. A CPU. CPU stands for ‘Central Processing Unit’. This little chip does all the work. It can do math, and move numbers around.

  2. Memory. Also called RAM, which stands for Random Access Memory. You can think of this is as the computer’s ‘short-term’ memory. Everything the computer needs while it’s running is here.

  3. Storage. We also call this the disk. When the computer needs to store something long-term, it gets put here.

Notice that I said ‘computer’, not ‘usable computer’. Your laptop or phone have screens, speakers, input devices. All of this stuff is extra, we need it to use the computer, but the computer doesn’t need it to compute.
